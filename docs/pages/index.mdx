### Description

This repository aims to centralise the current and future plugins I am working on, made by the community for the community.
That way, it seems easier for you to find them and contribute if you want.
All the plugins are under the MIT licence and are completely free. Please, do not hesitate to star the repository
in order to support the work. You can also [sponsor](https://github.com/sponsors/adrien2p)

here is the list of the plugins you can find in this documentation:

- [`medusa-xpo-auth-plugin`](/authentication)
- [`medusa-plugin-sentry`](/sentry)
- [`medusa-plugin-prometheus`](/prometheus)

### Contribute

This repository provides you with a set of plugins.

If you feel that you would like to contribute by creating a new plugin, integration or just report something,
then you can either:
- [create an issue](https://github.com/adrien2p/medusa-plugins/issues)
- [open a PR](https://github.com/adrien2p/medusa-plugins/pulls)
